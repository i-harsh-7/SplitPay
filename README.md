SplitPay

Splitting bills without splitting friendships

SplitPay is a smart expense-splitting mobile application that simplifies how groups manage shared expenses. Instead of manually calculating who owes what, SplitPay automatically extracts items from receipts, assigns them to users, and calculates fair splits.

The project combines OCR, LLM-powered parsing, and an intuitive mobile interface to make group payments seamless and transparent.  ￼

Problem

Splitting expenses sounds simple until real life happens.

Group dinners, trips, and shared purchases often create problems:
	•	Manual calculations are error-prone
	•	Equal splits are often unfair
	•	Existing apps require tedious manual entry
	•	Disagreements over who consumed what

These frictions lead to awkward conversations and delayed settlements.

SplitPay solves this with automated bill parsing and intelligent expense splitting.  ￼

⸻

Solution

SplitPay provides an AI-assisted workflow for splitting bills:
	1.	Upload or scan a receipt
	2.	OCR extracts the text
	3.	LLM converts raw text into structured items
	4.	Users assign items to group members
	5.	The system calculates fair splits automatically

The result is fast, transparent, and accurate expense sharing.

⸻

Key Features

Bill Scanning and OCR

Upload a receipt or take a photo.
The system extracts raw text using OCR.

AI-Powered Item Parsing

An LLM converts messy OCR output into structured bill items including:
	•	item name
	•	quantity
	•	price
	•	vendor
	•	total

Item-Level Expense Assignment

Users assign each item to one or more group members through a simple UI.

Multiple Split Methods

SplitPay supports:
	•	Equal split
	•	Per-item split
	•	Custom percentage split

Group Management

Create groups and manage members for events like:
	•	dinners
	•	trips
	•	shared shopping

Expense Tracking

Users can view:
	•	who owes money
	•	who is owed money
	•	historical expenses

Settlement Recording

Members can record payments and update balances instantly.
